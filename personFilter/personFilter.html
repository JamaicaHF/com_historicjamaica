Search by Name

















Search by Name allows you to use different techniques to find a person or family in the database.

"Last Starting With" will show a list of all people whose last name starts with the characters input.

"partial Name" allows you to enter the part of the name you know. For example, if you know only the first name, all people in the database whose first name match the name typed will become part of the result list.

"Similar Names" will try to look for people whose names are similar to the names entered.

"Last Name" will show women whose married or maiden name match the name entered.

We do not pretend to know all people who ever lived in Jamaica. We have entered all the vital records in our town records and tried to identify the people who are shown in our photo collection, but any genealogical information is not guaranteed to be correct, but only the best information available at the time.
